🔮 Mission 1: Neville's Remembrall

📜 The Story

Neville's grandmother sent him a Remembrall - a magical glass ball that turns red when you've forgotten something. But it's broken! It doesn't tell him WHAT he's forgotten.

Your mission: Use code magic to fix it by creating a todo list system!

🪄 Spell 1: Structurus (HTML)

First, we need to create the structure. In the magical world of code, HTML creates the bones of our spell:

<div id="remembrall" class="remembrall empty"></div> <input type="text" id="task" placeholder="What must you remember?"> <button onclick="addTask()">Cast Remembrall Charm</button> <ul id="tasks"></ul>

⚡ Spell 2: Dynamicus (JavaScript)

Now for the magic! JavaScript brings our Remembrall to life:

let tasks = []; function addTask() { const taskInput = document.getElementById('task'); const task = taskInput.value; if (task) { tasks.push(task); updateRemembrall(); taskInput.value = ''; } } function updateRemembrall() { const remembrall = document.getElementById('remembrall'); const taskList = document.getElementById('tasks'); if (tasks.length > 0) { remembrall.className = 'remembrall active'; taskList.innerHTML = tasks.map(t => `<li>${t}</li>`).join(''); } else { remembrall.className = 'remembrall empty'; taskList.innerHTML = ''; } }

🎯 Your Turn!

The code is ready, but Neville needs one more feature: the ability to mark tasks as complete!

Challenge: Add a function called completeTask() that removes a task when clicked.
